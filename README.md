# Vue-components
.vue单文件形式的组件，可直接导入使用；
也包含一些纯CSS控件

###已完成：
- 日期选择器：datePicker.vue
- 中国省市选择器：selectCity.vue
- 弹出提示框：beautyAlert.vue
- 图片裁剪并上传（简版）：crop.vue
- HTML5富文本编辑器：htmlEditor.vue
- Tab页：tab.vue


###开发中：
